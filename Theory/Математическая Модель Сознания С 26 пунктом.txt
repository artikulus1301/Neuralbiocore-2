Математическая модель сознания

---

1. Базовый нейронный уровень с фазовой динамикой

 Мембранный потенциал и фаза осциллятора:

τ_i · dV_i/dt = -V_i(t) + R_i · [Σ_j w_ij · s_j(t) + u_i(t)] + h_i(t) + σ_i · ξ_i(t)

Новое: Фазовая переменная для гамма-ритмов:

dφ_i/dt = ω_i + Σ_j K_ij · sin(φ_j - φ_i) + ε_φ · V_i(t)

Обозначения:
- `φ_i(t)` — фаза осциллятора нейрона i (0 до 2π)
- `ω_i` — собственная частота (~40 Гц для гамма-ритма)
- `K_ij` — сила фазовой связи
- `ε_φ` — связь амплитуды (мембранного потенциала) с фазой

Спайк с учетом фазы:

x_i(t) = f(V_i(t) - θ_i) · [1 + α_sync · cos(φ_i(t))]

Критерий синхронизации (order parameter Курамото):

R(t) · e^(iΨ(t)) = (1/N) · Σ_j e^(iφ_j(t))

Synchronized ⟺ R(t) > 0.8

---

2. Иерархическая архитектура с временными масштабами

τ_l · dX_l/dt = -X_l(t) + F_l(X_{l-1}(t), X_l(t), X_{l+1}(t), U_l(t), H_l(t))

Фазовая синхронизация между уровнями (cross-frequency coupling):

Φ_l(t) = ⟨φ_i(t)⟩_{i ∈ layer_l}

CFC_{l,l+1} = |⟨e^(i(n·Φ_l - Φ_{l+1}))⟩_t|  (n:1 фазовая связь)

--

3. Расширенная химическая модуляция

dH/dt = A(X(t)) - λ · H(t) + D(t)

Модуляция частоты осцилляций:

ω_i(t) = ω₀ + Δω_DA · DA(t) + Δω_ACh · ACh(t)

Пример:
- Дофамин (DA): ↑ DA → ↑ гамма-мощность
- Ацетилхолин (ACh): ↑ ACh → ↑ тета-ритмы (гиппокамп, 4-8 Гц)

Модуляция синаптической эффективности:

w_ij^{эфф}(t) = w_ij · m(H(t)) · [1 + κ · tanh(R_sync(t) - 0.5)]

(Синхронные коалиции усиливаются)

4. Пластичность с метапластичностью и консолидацией

 4a. STDP с фазовой зависимостью:

dw_ij/dt = η(θ_M) · [K(Δt, Δφ) · x_i(t) · x_j(t-Δt) - γ · w_ij · x_i²(t)] + ζ_ij(t)

Новое окно STDP с фазой:

K(Δt, Δφ) = A · exp(-Δt²/2τ²) · [1 + B · cos(Δφ)]

(Максимальная потенциация при Δφ ≈ 0, т.е. синфазные спайки)

4b. Метапластичность:

θ_M(t+1) = θ_M(t) + ε · [⟨x_i²⟩_T - θ_target]

4c. Консолидация во время сна

Фаза 1: Медленный сон (SWS)

if sleep_stage == 'SWS':
    D(t) → 0  (сенсорный вход отключен)
    
     Реактивация паттернов:
    X_replay(t) ~ p(X | w_current, history)
    
    Глобальная оптимизация весов:
    w_ij += η_sleep · ∫ [ε_l(X_replay) · ∂μ/∂w_ij] dt
    
     Снижение метапластического порога:
    θ_M → θ_M - Δθ_SWS

Фаза 2: REM-сон

if sleep_stage == 'REM':
     Генеративные "сны" (исследование латентного пространства):
    X_dream(t) ~ p_generative(X | w, H_random)
    
     Обучение генеративной модели:
    w_ij += η_REM · [∂log p(X_dream)/∂w_ij]
    
     Стирание временных связей (забывание):
    w_ij → w_ij · (1 - δ_forget · [1 - importance_ij])

Критерий важности синапса:

importance_ij = ⟨|∂L/∂w_ij|⟩_wakeful_experience

(Elastic Weight Consolidation — защита от катастрофического забывания)

5. Предиктивное кодирование + Активный вывод

 5a. Иерархическая генеративная модель:

Прогноз: μ_l(t) = g_l(X_{l+1}(t), a(t))  ← теперь зависит от действия!

Ошибка: ε_l(t) = Π_l^{1/2} · [X_{l-1}(t) - μ_l(t)]

Свободная энергия:
F = Σ_l ||ε_l||² + KL(q(X) || p(X)) + KL(q(a) || p(a))

 5b. Активный вывод (Action)

Минимизация свободной энергии через изменение мира:

da/dt = -∂F/∂a = -Σ_l [ε_l · ∂μ_l/∂a]

Интерпретация: Действие `a(t)` изменяет сенсорный вход `U(t+1)`, чтобы приблизить его к предсказаниям.

Проприоцептивный контур (рефлекс):

a_reflex(t) = -k_p · ε_proprio(t) - k_d · dε_proprio/dt

(PD-контроллер для минимизации ошибки между желаемой и текущей позой)

Целенаправленное действие:

a_goal(t) = arg min_a E[F(X_future | a, policy)]

policy = π(a | X, M)  ← зависит от мета-когниции

Исследование vs эксплуатация:

a_final(t) = (1 - β_explore) · a_goal(t) + β_explore · ξ_a(t)

β_explore ~ f(Θ)  ← выше при высокой неопределенности

6. Мета-когниция и самосознание

M(t+1) = r(M(t), X(t), a(t), Θ(t))

Θ_action(t) = Var[∂F/∂a]  ← неопределенность в выборе действия

Когнитивный конфликт:

C = ||M_predict - X_actual||² + ||a_intended - a_executed||²

Чувство свободы воли (Sense of Agency):

SoA(t) = exp(-λ · C_action(t))

C_action = ||outcome_predicted - outcome_actual||²

(Низкое расхождение → "я управляю своими действиями")

7. Глобальная интеграция с фазовой синхронизацией

7a. Конкурентная динамика коалиций:

dA_i/dt = -A_i/τ + Σ_j [W_ij · ρ(A_j)] + I_i(t) - β · Σ_{j≠i} A_j(t)

7b.  Фазовая синхронизация коалиций

+ γ_sync · Σ_j [A_j · cos(Φ_j - Φ_i)]

Обозначения:
- `Φ_i` — средняя фаза коалиции i
- `γ_sync` — вес синхронизации

Критерий сознательного доступа:

Conscious(t) = {i : A_i(t) > θ_A  AND  R_i(t) > θ_sync}

(Нужна и высокая активность, И синхронизация)

Трансляция в рабочую память:

dWM/dt = λ_in · Broadcasting(Conscious(t)) - λ_out · WM(t) - λ_interference · |Conflict(WM)|

8. Энергетический баланс и гомеостаз

8a. Метаболическая стоимость:

E(t) = Σ_i [c_spike · x_i(t) + c_synapse · Σ_j |w_ij · s_j(t)|] 
       + c_action · ||a(t)||² + c_baseline

8b. Циркадная модуляция:

H_circadian(t) = H₀ · [1 + A_circ · sin(2πt/24h + φ)]

if H_circadian < threshold:
    → Переход в сон (D(t) → 0, консолидация активируется)

8c. Гомеостатический драйв сна

S(t+1) = S(t) + Δt · [k_wake · awake(t) - k_sleep · asleep(t)]

P(sleep | S) = sigmoid(α · (S - S_threshold))

Интерпретация: Накопление "долга сна" → увеличение вероятности засыпания.

9. Патологические состояния

 9a. Существующие:

- Депрессия: DA ↓, 5-HT ↓ → пессимистичные прогнозы
- Тревожность: Π(t) ↑↑ → гиперточность к ошибкам
- Шизофрения: Π нарушена, DA ↑ → галлюцинации

 9b. Дополнение:

Бессонница:

S(t) ↑↑  but  P(sleep) ↓  (нарушение гомеостата)
→ Накопление ошибок, ↓ консолидация, когнитивный дефицит

Болезнь Паркинсона:

DA ↓↓ → ↓ ω_gamma, ↓ фазовая синхронизация
→ Замедление действий (bradykinesia), тремор (нарушение рефлексов)

Эпилепсия:

K_ij ↑↑ → гиперсинхронизация (R(t) → 1)
→ Каскадная активация, судороги

Кома:

R(t) → 0  (десинхронизация)  OR  R(t) → 1, ω → 0  (замедленные волны)
Φ(X) ≈ 0  (потеря интеграции)

10. Критерий сознательности

10a. Интегрированная информация (IIT):

Φ(X) = min_partition MI(X₁ : X₂ | cut)

Φ_eff(X) = Φ(X) · f(R_sync(t))  ← учет синхронизации

10b. Причинная плотность:

CD(t) = Rank[G(X_t, X_{t-τ})]  ← Granger causality matrix

10c. Критерий сознания:

Conscious ⟺ Φ_eff > Φ_min  AND  R_sync > 0.7  AND  WM_load > 0

11. Сон: циклы и функции

Модель сна/бодрствования:

State = {Wake, N1, N2, N3 (SWS), REM}

Transitions:
Wake → N1 → N2 → N3 → REM → N2 → ... (циклы 90 мин)

Во время SWS (медленный сон):

- Частота: δ-волны (0.5-4 Гц), ω_i → 1-2 Гц
- Активность: A_i ↓↓, R_sync → высокая (гиперсинхронизация)
- Функция: Консолидация декларативной памяти
- Процесс:
  
  for epoch in SWS:
       Реактивация гиппокампальных паттернов:
      X_replay ~ sharp-wave ripples (150-250 Гц)
      
       Перенос в неокортекс:
      w_cortex += η · ⟨X_hippocampus(t-δ) · X_cortex(t)⟩
      
      Синаптический даунскейлинг:
      w_ij → w_ij · (1 - δ_SWS · [1 - importance_ij])

Во время REM:

- Частота: θ (4-8 Гц), β (15-30 Гц), гамма
- Активность: A_i высокая, но D(t) = 0 (сенсорный вход отключен)
- Функция: Обучение генеративной модели, процедурная память, забывание
- Процесс:
  
  for epoch in REM:
       Генерация "снов":
      X_dream(t) ~ p(X | w, H_random, a_random)
      
      Обучение предсказаниям:
      Minimize: F_generative = -log p(X_dream | w)
      
      Стирание слабых связей:
      w_ij → w_ij - η_forget · (1 - importance_ij)
      
      Консолидация моторных навыков:
      w_motor += η_REM · ⟨ε_action · ∂μ/∂w⟩

Депривация сна:

if sleep_debt > critical:
    → θ_M ↑ (снижение пластичности)
    → w_ij нерегуляризованные (переобучение)
    → Π(t) ↑ (тревожность)
    → Φ(X) ↓ (микросны, фрагментация сознания)

12. Проблема связывания (Binding Problem) — решение

12a. Механизм связывания через синхронизацию:

Гипотеза: Объекты/признаки связываются через гамма-синхронизацию (~40 Гц) нейронов в разных модулях.

Математика:

Для объекта O_k:

Neurons_k = {i : feature_i ∈ O_k}

Binding(O_k, t) ⟺ |⟨e^(iφ_i(t))⟩_{i ∈ Neurons_k}| > 0.9

Пример: Красный квадрат

- V4 (цвет): нейроны "красный" → φ_color(t)
- V1 (форма): нейроны "квадрат" → φ_shape(t)
- Связывание: |φ_color - φ_shape| < π/4 (синхронны)

12b. Уравнение связывания в GWT:

dA_i/dt = ... + γ_bind · Σ_{j ∈ related} [A_j · cos(Φ_j - Φ_i) · semantic_similarity(i,j)]

Результат: Семантически связанные коалиции синхронизируются и выигрывают конкуренцию.

13. Активный вывод: замыкание контура

 Полная система "мозг ↔ тело ↔ среда":

dX/dt = f_perception(U, X, a)  ← восприятие
da/dt = f_action(X, M, Θ)      ← действие
dU/dt = f_environment(U, a)    ← среда меняется от действий

Свободная энергия (полная):

F_total = F_perception + F_action + F_proprioception

где:
F_action = ||a_predicted - a_executed||² / σ_a²
F_proprio = ||body_state_predicted - body_state_actual||² / σ_p²
Минимизация:

∂X/∂t = -∂F/∂X  ← обновление убеждений (восприятие)
∂a/∂t = -∂F/∂a  ← выбор действия (моторный контроль)

Эквивалентность:

Активный вывод = RL (reinforcement learning), где:

Reward = -F
Value function = -E[F_future]
Policy = arg min_π E[F | π]

14. Смерть и прекращение сознания

 Каскад разрушения:
Стадия 1: Аноксия

I_metabolic → 0
E_available(t) → E_available(t-1) - E(t)  (запасы истощаются)

Стадия 2: Энергетический кризис

if E_available < E_critical:
    ATP ↓↓
    Na⁺/K⁺-ATPase fails
    → Деполяризация: V_i → V_rest + 30 mV

Стадия 3: Эксайтотоксичность

Ca²⁺ influx → активация протеаз → синаптическая деградация
w_ij(t+1) ~ w_ij(t) + Huge_noise

Стадия 4: Потеря синхронизации

R(t) → 0  (нейроны разряжаются хаотично)
Φ(X) → 0  (нет интеграции)

Стадия 5: Электрическое молчание

x_i(t) → 0  для всех i
X(t) → шум
M(t) → undefined

Критерий необратимой смерти:

∫_{t_0}^{t_0+10min} ||X(t)||² dt ≈ 0  AND  ∂w_ij/∂t ≈ 0

(Отсутствие активности и невозможность восстановления синаптических связей)

 17. GABAergic модуляция и анестезия

17a. Ингибиторная динамика

GABA-рецепторы (GABA_A и GABA_B):

I_GABA_i(t) = g_GABA_A · (V_i - E_Cl) · s_GABA_A(t) 
            + g_GABA_B · (V_i - E_K) · s_GABA_B(t)

Обозначения:
- `g_GABA_A/B` — проводимость GABA-рецепторов
- `E_Cl` ≈ -70 мВ, `E_K` ≈ -90 мВ (реверсивные потенциалы)
- `s_GABA(t)` — синаптическая переменная

Синаптическая динамика GABA:

```
ds_GABA_A/dt = α_A · [GABA](t) · (1 - s_GABA_A) - β_A · s_GABA_A
ds_GABA_B/dt = α_B · [GABA](t) · (1 - s_GABA_B) - β_B · s_GABA_B
```

Временные константы:
- GABA_A: быстрая (τ ~ 10-20 мс) — фазическая ингибиция
- GABA_B: медленная (τ ~ 100-300 мс) — тоническая ингибиция

 17b. Обновленное уравнение нейрона:

```
τ_i · dV_i/dt = -V_i(t) 
                + R_i · [Σ_j w_ij^exc · s_j^exc(t) - Σ_k w_ik^inh · s_k^inh(t) + u_i(t)]
                - I_GABA_i(t)
                + h_i(t) 
                + σ_i · ξ_i(t)
```

17c. Интернейроны (PV, SST, VIP)

Три класса ингибиторных интернейронов:

1. Parvalbumin+ (PV) — быстрые, перисоматические:
```
τ_PV · dV_PV/dt = -V_PV + Σ_j w_PV,j · x_j(t)

x_PV(t) = f_fast(V_PV - θ_PV)  # высокочастотные спайки

Мишени: сома пирамидальных нейронов
Функция: winner-take-all, гамма-ритмы
```

2. Somatostatin+ (SST) — дендритные:
```
τ_SST · dV_SST/dt = -V_SST + Σ_j w_SST,j · x_j(t) + top_down(X_{l+1})

x_SST(t) = f_slow(V_SST - θ_SST)

Мишени: апикальные дендриты
Функция: модуляция top-down сигналов
```

3. Vasoactive Intestinal Peptide+ (VIP) — дизингибиция:
```
VIP → SST ⊣  (ингибирует SST)
VIP → PV ⊣

Результат: растормаживание (disinhibition)
Функция: усиление сигнала, внимание
```

Микросхема:

```
Excitatory (E) → PV → E  (feed-forward inhibition)
E → SST → E_dendrites     (modulatory inhibition)
E → VIP → SST → E         (disinhibition)
```

17d. Анестезия: механизм действия

Пропофол, Изофлуран, Севофлуран:

Усиливают GABA_A рецепторы:

```
g_GABA_A^{anes}(t) = g_GABA_A · [1 + k_anes · [Anesthetic](t)]

где k_propofol ≈ 5-10
```

Эффект на систему:

```
1. Увеличение тонической ингибиции:
   V_i → более гиперполяризованное состояние
   
2. Снижение возбудимости:
   x_i(t) ↓↓
   
3. Разрыв таламо-кортикальных петель:
   Thalamus ⇄ Cortex связь ослабевает
   
4. Разрушение фазовой синхронизации:
   R(t) → 0  (нейроны не могут синхронизироваться)
   
5. Фрагментация глобального workspace:
   A_i(t) < θ_consciousness для всех i
   
6. Коллапс интеграции:
   Φ(X) → 0
```
Уравнение под анестезией:

```
dX_l/dt = -X_l/τ_l + F_l(...) - K_anes · [Anes](t) · X_l

где K_anes → ∞ при глубокой анестезии
```

Стадии анестезии:

```
[Anes]    Эффект                           Φ(X)    R_sync
------------------------------------------------------------------------
0-1 μM    Седация, сонливость              0.4     0.6
1-3 μM    Потеря сознания, амнезия         0.1     0.3
3-5 μM    Хирургическая анестезия          0.01    0.1
>5 μM     Подавление стволовых рефлексов   ~0      ~0
```

Пробуждение (emergence):

```
d[Anes]/dt = -λ_clearance · [Anes](t)

По мере клиренса:
1. Thalamus reactivates
2. R_sync восстанавливается (α → β → γ ритмы)
3. A_i превышает порог → сознание возвращается
4. Φ(X) растет

Критический момент: Φ(X) > Φ_threshold
```

Гистерезис:

Засыпание ≠ пробуждение (разные пороги из-за адаптации рецепторов):

```
[Anes]_sleep > [Anes]_wake
```

18. Аффективное измерение — нейробиология эмоций

 18a. Ключевые структуры

Миндалина (Amygdala):
```
A(t) = [A_LA(t), A_BLA(t), A_CeA(t)]ᵀ

LA  — латеральная (сенсорный вход)
BLA — базолатеральная (оценка значимости)
CeA — центральная (висцеральный выход)
```

Островок (Insula):
```
I(t) = [I_posterior(t), I_anterior(t)]ᵀ

Posterior: интероцепция (сердцебиение, дыхание, боль)
Anterior: субъективное чувство (интеграция в сознание)
```

Вентральное mPFC (vmPFC):
```
vmPFC(t) — оценка ценности (value), регуляция эмоций
```

Вентральный стриатум (Ventral Striatum / NAcc):
```
VS(t) — предвкушение награды, мотивация
```

18b. Двумерная модель аффекта (Russell's Circumplex)

Валентность (Valence) и Возбуждение (Arousal):

```
Emotion(t) = [V(t), A(t)]ᵀ

V(t) ∈ [-1, 1]  (негативное ← → позитивное)
A(t) ∈ [0, 1]   (спокойное → возбужденное)
```

Примеры:
- Радость: V=+0.8, A=0.7
- Гнев: V=-0.6, A=0.9
- Грусть: V=-0.7, A=0.3
- Спокойствие: V=+0.3, A=0.1

 18c. Вычисление валентности и возбуждения

Валентность:

```
dV/dt = (1/τ_V) · [μ_V - V(t)] + β_reward · δ_RPE(t) + β_pain · Pain(t)

где:
μ_V(t) = f_vmPFC(X_context(t), Goals(t))  — базовый тон
δ_RPE(t) = Reward(t) - Prediction(t)      — ошибка предсказания награды
```
Возбуждение:

```
dA/dt = (1/τ_A) · [-A(t)] + α_surprise · |ε_total(t)| + α_threat · Threat(t)
                           + α_NE · NE(t) + α_insula · I(t)

где:
|ε_total| = Σ_l ||ε_l(t)||  — суммарная ошибка предсказания
Threat(t) = A_CeA(t)         — активность миндалины
```

 18d. Эмоциональная динамика миндалины

Латеральная миндалина (сенсорный вход):

```
dA_LA/dt = Σ_sensory w_sen · x_sensory(t) - λ_LA · A_LA(t)
```

Базолатеральная (условный стимул):

```
dA_BLA/dt = w_LA→BLA · A_LA(t) + w_context · X_hippocampus(t) 
           - w_vmPFC→BLA · vmPFC(t)  ← регуляция сверху вниз
           - λ_BLA · A_BLA(t)
```

Центральная (выход к телу):

```
dA_CeA/dt = w_BLA→CeA · A_BLA(t) - λ_CeA · A_CeA(t)

Outputs:
- HPA axis → кортизол (стресс)
- Симпатическая НС → адреналин (arousal)
- PAG → замораживание/бегство
- Лицевой нерв → экспрессия эмоций
```

 18e. Островок (Insula) — интероцепция

Задний островок (прямой сигнал от тела):

```
dI_post/dt = Σ_i w_i · Body_signal_i(t) - λ_post · I_post(t)

Body_signals: {
  HR(t)        — частота сердцебиения
  BP(t)        — кровяное давление  
  Resp(t)      — дыхание
  Pain(t)      — ноцицепция
  Temp(t)      — температура
  GI(t)        — желудочно-кишечные
}
```

Передний островок (субъективные чувства):

```
dI_ant/dt = w_post · I_post(t) + w_context · X(t) - λ_ant · I_ant(t)

Feeling(t) = f(I_ant(t), V(t), A(t))
```

Интероцептивная точность:

```
IA(t) = Corr(I_post(t), Body_true(t))

Высокая IA → лучшее осознание эмоций (alexithymia при низкой IA)
```

18f. Эмоциональная модуляция познания

Влияние на свободную энергию:

```
F_emotion = F_cognitive + λ_affect · E_affect

где:
E_affect = -V(t) · Outcome(t) + A(t) · Uncertainty(t)
```

Модуляция обучения:

```
η_effective(t) = η · [1 + β_arousal · A(t)] · sign(V(t))

Высокое возбуждение → быстрее обучение
Негативная валентность → избегание
```

Модуляция внимания:

```
Π_l(t) = Π_baseline · [1 + γ_emotion · A_BLA(t)]

Угроза → усиление точности (гиперчувствительность)
```

Модуляция синхронизации:

```
ω_i(t) = ω_baseline + Δω_arousal · A(t)

Высокое возбуждение → ↑ гамма-частота (усиление обработки)
```

18g. Специфические эмоции через паттерны активации

Страх:
```
Fear(t) ∝ A_CeA(t) · [1 - vmPFC(t)]

Body: ↑HR, ↑BP, потоотделение, мидриаз
Action: замораживание или бегство (a_freeze или a_escape)
Cognition: Π ↑ (гиперчувствительность к угрозам)

Гнев:

Anger(t) ∝ A_CeA(t) · Frustration(t) · Testosterone(t)

Frustration = |Goal - State|² при высокой Arousal
Body: ↑BP, ↑мышечное напряжение
Action: агрессия (a_attack)

Радость:

Joy(t) ∝ VS(t) · δ_RPE(t)  при δ_RPE > 0

VS: ventral striatum (предвкушение + получение награды)
Body: ↑дофамин, улыбка
Action: подход (a_approach)

Грусть:

Sadness(t) ∝ -V(t) · Low_A(t) · [5-HT ↓]

Body: ↓энергия, плач
Action: withdrawal (социальная изоляция)
Cognition: пессимистичные прогнозы (μ_V < 0)

Отвращение:

Disgust(t) ∝ I_ant(t) · Contamination_signal(t)

Body: тошнота, активация GI
Action: avoidance, отвержение
Функция: защита от патогенов/токсинов

18h. Регуляция эмоций (vmPFC → Amygdala)

Когнитивная переоценка (reappraisal):

Target: уменьшить A_BLA через изменение интерпретации

w_vmPFC→BLA(t+1) = w_vmPFC→BLA(t) + η_reg · [Goal_emotion - Emotion_current]

Стратегии:
1. Реинтерпретация: "не угроза, а вызов"
2. Дистанцирование: "наблюдаю со стороны"
3. Подавление экспрессии: блокировка A_CeA → face muscles

Подавление vs Переоценка:

Suppression: блокирует выход (A_CeA → body), но A_BLA остается высокой
  → затратно энергетически
  → физиологический стресс сохраняется

Reappraisal: уменьшает A_BLA на входе
  → более эффективно
  → меньше затрат

18i. Эмоции в сознании

Интеграция в Global Workspace:

Emotional_content(t) = [V(t), A(t), Feeling(t), Body_state(t)]

Conscious_emotion ⟺ Emotional_content ∈ Workspace AND Φ > threshold

Чувство (Feeling) vs Эмоция (Emotion):

Emotion: Body(t) + A_CeA(t)  ← бессознательная реакция
Feeling: I_ant(t) + Workspace ← осознанное переживание

Feeling = Interoception + Context + Self-awareness

Алекситимия (неспособность описать эмоции):

I_ant ⫫ Workspace  (островок не интегрирован)

→ Чувствуют телесные реакции, но не могут назвать эмоцию

19. Theory of Mind — рекурсия второго порядка

19a. Базовая структура

Уровни ментализации:

Level 0: Self-model
  M₀(t) = r(X(t), a(t), Θ(t))
  
Level 1: Other-model
  M₁_other(t) = r_ToM(O_observable(t), Context(t))
  
Level 2: Self-modeling-other-modeling-self
  M₂_self(t) = "Что он думает обо мне?"
  M₂_self(t) = r_ToM(M₁_other(t), My_actions(t))

Обозначения:
- M₀ — моя модель себя
- M₁_other — моя модель другого человека
- M₂ — моя модель того, что другой думает обо мне
- r_ToM — функция менталистического вывода

19b. Математическая формализация

Модель другого как генеративная модель:

p(O_behavior | M_other) = ∫ p(O | a, X_other) · p(a | Goals_other, X_other) 
                            · p(X_other | M_other) dX_other da

Обратный вывод (inverse inference):

p(M_other | O_behavior) ∝ p(O_behavior | M_other) · p(M_other)

Т.е., наблюдая поведение O, делаем вывод о ментальном состоянии другого

19c. Компоненты ToM

1. Убеждения другого (Beliefs):

B_other(t) = {что он знает, во что верит}

dB_other/dt = f_update(O_other_observable(t), History)

Пример: False Belief Task (Салли-Энн)
  B_self: мяч в корзине B
  B_Sally: мяч в корзине A (она не видела перемещение)

2. Желания другого (Desires/Goals):

G_other(t) = arg max_{g} p(a_other | g, B_other)

Вывод через Inverse RL:
  Наблюдая действия a_other(t), делаем вывод о вознаграждении R_other(s)

3. Намерения другого (Intentions):

I_other(t) = p(a_other_future | G_other, B_other, X_other)

"Что он собирается сделать дальше?"

4. Эмоции другого:

E_other(t) = [V_other(t), A_other(t)]

Вывод через:
- Наблюдение лицевой экспрессии: O_face(t) → E_other
- Контекст: X_context → E_other  
- Теория: "если его цель заблокирована → Anger_other ↑"

19d. Нейронные субстраты ToM

Ключевые области:

1. Temporo-Parietal Junction (TPJ):
   - Обработка перспективы другого
   - "Switch" между self и other

2. Medial Prefrontal Cortex (mPFC):
   - Ментальные состояния (убеждения, желания)
   - Рекурсия: "он думает, что я думаю..."

3. Posterior Superior Temporal Sulcus (pSTS):
   - Биологическое движение
   - Намерения из действий

4. Precuneus:
   - Self-other distinction
   - Эпизодическая память (симуляция другого)

Зеркальные нейроны:

Mirror neurons: активны как при выполнении, так и при наблюдении действия

x_i^{mirror}(t) = f(a_self(t)) + f(O_other_action(t))

Функция: прямое понимание моторных намерений
  "Я вижу, что он тянется к чашке → он хочет пить"

19e. Математическая модель рекурсии

Уровень 0: Self-model

```
dM₀/dt = r₀(X_self(t), a_self(t), Body_self(t), Θ_self(t))

M₀ = {
  X_self:       мои перцепты
  a_self:       мои действия
  G_self:       мои цели
  E_self:       мои эмоции
  B_self:       мои убеждения
}

Уровень 1: Model of Other

dM₁_other/dt = r₁(O_other(t), Context(t), M₀(t), Priors_social)

M₁_other = {
  X_other:      что он воспринимает (по моей оценке)
  a_other:      его действия (наблюдаемые)
  G_other:      его цели (инференция)
  E_other:      его эмоции (через эмпатию или теорию)
  B_other:      его убеждения (могут отличаться от B_self!)
}

Критически: B_other ≠ B_self  (понимание false beliefs)

Уровень 2: Model of Other's Model of Self

dM₂_self_in_other/dt = r₂(M₁_other(t), a_self(t), O_self_visible(t))

M₂_self_in_other = "Что он думает обо мне?"

M₂ = {
  X_self_as_seen_by_other:  как он видит меня
  G_self_as_inferred:       какие цели он мне приписывает
  E_self_as_perceived:      какие эмоции он во мне считывает
}

Уровень 3: ... и выше (рекурсия)

M₃ = "Что я думаю о том, что он думает о том, что я думаю о нем"

В теории: бесконечная рекурсия
На практике: люди ограничены ~3-5 уровнями рекурсии
  (когнитивная нагрузка, рабочая память)

19f. Уравнения рекурсивного обновления

Общая форма:

M_n+1(t) = r_ToM(M_n(t), O_level_n(t), Context(t))

где O_level_n — наблюдаемые на уровне n (действия, слова, экспрессия)

Свободная энергия для ToM:

F_ToM = Σ_n [||O_level_n - μ_n(M_n)||² / σ_n² + KL(q(M_n) || p(M_n))]

Минимизация через:
∂M_n/∂t = -∂F_ToM/∂M_n

Интерпретация:
- Подгоняем модель другого, чтобы его поведение было предсказуемо
- Баланс между данными (O) и априорными ожиданиями p(M)

19g. Эмпатия vs Theory of Mind

Эмпатия (аффективная):

E_self(t) ← E_other(t)  (эмоциональное заражение)

dE_self/dt = κ_empathy · [E_other(t) - E_self(t)]

Механизм: зеркальные нейроны + островок + ACC
  "Я чувствую твою боль"

Theory of Mind (когнитивная):

Inference: O_other → M_other  (рассудочное понимание)

"Я понимаю, что ты грустен, но не обязательно чувствую грусть"

Расстройства:

Autism Spectrum Disorder:
  - ToM нарушена (сложности с B_other, G_other)
  - Эмпатия может быть сохранна или усилена

Psychopathy:
  - ToM сохранна (манипуляции)
  - Аффективная эмпатия снижена (отсутствие сопереживания)

19h. Социальные взаимодействия

Игра двух агентов:

Agent A:
  M_A = self-model
  M_A→B = model of B
  M_A→B→A = "что B думает обо мне"

Agent B:
  M_B = self-model
  M_B→A = model of A
  M_B→A→B = "что A думает обо мне"

Взаимное обновление:

dM_A→B/dt = f(O_B(t), M_A, Communication_B→A)
dM_B→A/dt = f(O_A(t), M_B, Communication_A→B)

Communication: язык, жесты, экспрессия

Общее знание (Common Ground):

CG(t) = M_A ∩ M_B ∩ "A знает, что B знает" ∩ "B знает, что A знает" ∩ ...

Необходимо для успешной коммуникации

19i. Ложь и обман

Модель намеренной дезинформации:

Agent A хочет создать ложное M_B→A:

1. A вычисляет истинное M_A
2. A создает "показное" поведение O_A_fake
3. O_A_fake → M_B→A_false (B делает неверный вывод)
4. A использует M_B→A_false для манипуляции

Требует:
- Работающий ToM (предсказать реакцию B)
- Ингибиция правдивого поведения
- Рабочая память (удержать истину и ложь раздельно)

Обнаружение лжи:

Inconsistency: O_A_verbal ≠ O_A_nonverbal

B замечает: микроэкспрессии, когнитивную нагрузку у A
  → "Что-то не так, его модель не согласована"

19j. Развитие ToM

Онтогенез:

Возраст 1-2 года: Joint attention (совместное внимание)
  "Мы оба смотрим на одно и то же"

Возраст 3-4 года: False Belief понимание
  "Салли думает, что мяч в корзине A"

Возраст 5-6 лет: Recursive ToM (2 уровня)
  "Он думает, что я не знаю"

Взрослость: 3-5 уровней рекурсии

Критические факторы:

1. Prefrontal cortex созревание
2. Рабочая память (WM capacity)
3. Ингибиция (подавить собственную перспективу)
4. Социальный опыт (обучение через взаимодействие)

 19k. Интеграция ToM с эмоциями

Эмпатическая дистресс:

```
if E_other = Fear AND Similarity_other_to_self > threshold:
    E_self ← E_other  (эмоциональное заражение)
    
    if vmPFC_regulation:
        → Compassion (сострадание, но без дистресса)
    else:
        → Personal distress (собственный дистресс, избегание)

Социальная боль:

Rejection(t) = M₂_self_in_other показывает "они меня не ценят"

→ Активация: ACC (anterior cingulate cortex), такая же как физическая боль
→ E_self = Sadness, V ↓

Вина и стыд:

Guilt = "Мои действия причинили вред другому"
  = f(a_self, E_other_negative, Responsibility_self)

Shame = "Я плох в глазах других"
  = f(M₂_self_in_other_negative, Self-concept_discrepancy)

19l. ToM в сознании

Критерий сознательного ToM:

Conscious_ToM ⟺ M₁_other ∈ Workspace AND Φ(M₁) > threshold

Можно иметь автоматический ToM (implicit) без осознания:
  - Быстрые социальные реакции
  - "Чувствую, что что он собирается сделать, но не могу объяснить почему"

Осознанная рефлексия:

Metacognitive_ToM(t) = r(M₁_other(t), Θ_ToM(t))

Θ_ToM = неопределенность в модели другого

"Насколько я уверен в том, что я правильно понял его намерения?"

Перспективное взятие (Perspective Taking):

Switch_mode:
  if mode == "Self":
      Active_model = M₀
      Inhibit(M₁_other)
  
  if mode == "Other":
      Active_model = M₁_other
      Inhibit(M₀)  ← требует исполнительного контроля

Difficulty: подавить эгоцентрическую перспективу (egocentric bias)

20. Интеграция: Эмоции × ToM × Сознание × Анестезия

20a. Полная архитектура системы

Level 0: Нейроны + GABA
  ↓
Level 1: Модули (V1, V2, Amygdala, Insula, mPFC, TPJ)
  ↓
Level 2: Интеграция (Global Workspace + Фазовая синхронизация)
  ↓
Level 3: Сознание (Φ, Self-model M₀)
  ↓
Level 4: Эмоции (V, A, Feelings)
  ↓
Level 5: Social Cognition (M₁_other, M₂, ...)

20b. Объединенное уравнение состояния

dS/dt = F_dynamics(S, U, H, GABA, Emotion, ToM, Anesthesia)

где S = [X, M₀, M₁, M₂, V, A, Body_state, φ, w, ...]ᵀ

Компоненты:

1. Сенсорно-моторный контур:
   dX/dt = f_perception(U, X, a, GABA)
   da/dt = f_action(X, M₀, E)
   
2. Мета-когниция:
   dM₀/dt = r₀(X, a, Body, Θ)
   
3. Эмоции:
   dV/dt = (μ_V - V)/τ_V + β·δ_RPE + γ·Threat
   dA/dt = -A/τ_A + α·|ε| + α_NE·NE + α_threat·A_CeA
   
4. Тело (интероцепция):
   dBody/dt = f_physiology(a, E, Stress)
   
5. ToM:
   dM₁/dt = r₁(O_other, Context, M₀)
   dM₂/dt = r₂(M₁, a_self, O_self_visible)
   
6. Химия:
   dH/dt = A(X, E) - λ·H + D
   d[GABA]/dt = Production - Uptake + [Anesthetic]·κ
   
7. Синхронизация:
   dφ/dt = ω + K·sync(φ_neighbors) + ε·V
   
8. Пластичность:
   dw/dt = η(θ_M)·STDP(x, Δt, Δφ) - γ·w

20c. Анестезия подавляет все уровни

[Anesthetic](t) ↑
  ↓
GABA_A ↑↑ → V_i гиперполяризация
  ↓
x_i ↓ → X ↓
  ↓
φ десинхронизация → R → 0
  ↓
A_i < θ → Global Workspace коллапс
  ↓
Φ(X) → 0
  ↓
M₀ → undefined (потеря самосознания)
  ↓
E_self → ? (чувства исчезают)
  ↓
M₁_other → неактивен (ToM отключается)

Порядок отключения:

Легкая седация:
  M₂, M₁ ↓ (социальное познание страдает первым)
  Эмоциональная регуляция ↓
  
Умеренная анестезия:
  M₀ фрагментируется (disorientation)
  WM collapse
  V, A притупляются
  
Глубокая анестезия:
  X → минимальная обработка
  Φ → 0
  Полная потеря сознания
  
Очень глубокая:
  Стволовые рефлексы подавлены
  Дыхание требует поддержки

 20d. Эмоциональная модуляция ToM

High Arousal + Negative Valence (стресс):
  → Egocentric bias ↑ (сложнее принять чужую перспективу)
  → M₁_other менее точный
  → Враждебная атрибуция: "Он сделал это специально" (даже если нет)

High Arousal + Positive Valence (возбуждение):
  → Prosocial behavior ↑
  → Эмпатия ↑
  
Low Arousal + Negative Valence (грусть):
  → Rumination on M₂ ("Что они обо мне думают?")
  → Social withdrawal

Математически:

Accuracy_ToM(t) = Baseline_accuracy · [1 - α·|V(t) - V_optimal|] 
                                    · [1 - β·(A(t) - A_optimal)²]

V_optimal ≈ 0.2 (слегка позитивный)
A_optimal ≈ 0.5 (умеренное возбуждение)

Экстремальные эмоции → плохой ToM

 20e. Social Pain → GABA дисрегуляция

Chronic social rejection:
  → хронический стресс
  → кортизол ↑↑
  → GABA_A рецепторы down-regulated
  → гиперчувствительность к угрозам
  → тревожность
  → избегание социальных ситуаций
  → еще больше rejection
  
Порочный круг:
  Rejection → Anxiety → Social avoidance → Rejection

Терапевтические мишени:

1. Фармакологические:
   - Бензодиазепины (краткосрочно): ↑ GABA → ↓ тревожность
   - SSRI: ↑ серотонин → ↓ A_CeA → регуляция эмоций
   
2. Когнитивные:
   - Reappraisal training: улучшить vmPFC → Amygdala контроль
   - ToM тренировка: точнее M₁_other → меньше враждебных атрибуций
   
3. Социальные:
   - Социальная поддержка: положительные взаимодействия → ↑ окситоцин
   - Mindfulness: улучшить интероцепцию (I_ant) → лучшая эмоциональная осведомленность

21. Патологии с новыми компонентами

 21a. Социальная тревожность

Система:
  M₂_self_in_others = катастрофически негативная
  "Все думают, что я странный/неумелый"
  
  → A ↑↑ (высокое возбуждение в социальных ситуациях)
  → V ↓ (негативная валентность)
  → Π_social ↑↑ (гиперчувствительность к социальным сигналам)
  → A_CeA ↑ (воспринимаемая угроза)
  
Механизм:
  - Ошибочный ToM: переоценка негативных суждений других
  - GABA дисрегуляция → недостаточная ингибиция миндалины
  - Избегание → нет коррекции ошибочных моделей M₂

Лечение:

CBT (Cognitive Behavioral Therapy):
  - Исправить M₂: "Они не думают обо мне так плохо"
  - Экспозиция: тестировать M₂ в реальности → обновление
  
SSRI:
  - ↑ 5-HT → ↓ A_CeA → меньше воспринимаемой угрозы

 21b. Расстройство личности (Borderline PD)

Характеристики:
  - Эмоциональная дисрегуляция: V(t), A(t) крайне нестабильны
  - Проблемы с привязанностью: M₁_other нестабильный
  - "Splitting": M₁_other переключается между "идеализация" и "обесценивание"
  
Механизм:
  - Гиперактивная миндалина (A_BLA, A_CeA ↑↑)
  - Слабый vmPFC → Amygdala контроль
  - Нарушенная интероцепция (I_ant)
  
Эмоциональная формула:
  dV/dt имеет большой коэффициент → быстрые свинги
  dA/dt = α_huge · |ε| → сильные реакции на малые стимулы

Терапия:

DBT (Dialectical Behavior Therapy):
  - Mindfulness → улучшить I_ant (осознание эмоций)
  - Emotion regulation skills → усилить vmPFC контроль
  - Interpersonal effectiveness → стабилизировать M₁_other

21c. Аутизм: ToM и сенсорная обработка

ToM дефицит:
  - M₁_other менее точный (сложности с B_other, G_other)
  - M₂ практически недоступен
  - Трудности с неявными социальными правилами
  
Сенсорная чувствительность:
  - Π_sensory ↑↑ (гиперчувствительность)
  - σ_noise ↑ (больший внутренний шум)
  - Сложности с фильтрацией (слабые PV интернейроны?)
  
Компенсация:
  - Использование явных правил вместо интуитивного ToM
  - "Я выучил, что когда люди хмурятся, они недовольны"
  - Системное мышление (strengths в паттернах, слабость в социальном)

21d. Психопатия: ToM без эмпатии

Когнитивный ToM: сохранен (часто даже усилен)
  - M₁_other точный → эффективная манипуляция
  - M₂ используется для обмана
  
Аффективная эмпатия: нарушена
  - κ_empathy ≈ 0  (нет эмоционального заражения)
  - Островок (insula) гипоактивен
  - vmPFC → Amygdala связи слабые
  
Результат:
  - "Я знаю, что ты чувствуешь (ToM), но мне все равно (no empathy)"
  - Инструментальная агрессия (для достижения целей)
  - Отсутствие вины: M₂ не генерирует "они страдают по моей вине"

21e. Депрессия: эмоции + ToM + анестезия параллель

Эмоциональный профиль:
  - V(t) << 0 (стабильно негативная валентность)
  - A(t) низкая (ангедония, усталость)
  - μ_V сдвинут вниз (пессимистичные прогнозы)
  
ToM нарушения:
  - M₂_self_in_others = "Я бремя для других"
  - Социальная изоляция → меньше обновлений M₁
  
Нейрохимия:
  - DA ↓, 5-HT ↓, NE ↓
  - Похоже на "химическую анестезию" мотивации
  
Феноменология:
  - Мир воспринимается через "серый фильтр"
  - Действия кажутся бессмысленными (a → ничего не меняет)
  - Время течет медленно (как в измененных состояниях сознания)
  
Параллель с анестезией:
  - Не полная потеря сознания, но "притупление" (dampening)
  - Φ(X) снижена (меньше интеграции)
  - R_sync может быть нарушена

Лечение:

Антидепрессанты:
  - SSRI: ↑ 5-HT → постепенно сдвиг μ_V вверх
  - NDRI: ↑ DA, NE → ↑ мотивация, энергия
  
Психотерапия:
  - CBT: исправить пессимистичные прогнозы μ_V
  - Behavioral activation: a → положительный исход → обновить модель
  
Нейростимуляция:
  - ECT: "reset" → изменение синаптических весов w
  - TMS: таргетная стимуляция dlPFC

22a. Эмоциональная обработка во сне

During REM:

  2. Интеграция эмоционального опыта:
     "Сны" как генеративная симуляция эмоциональных сценариев
  
  3. Регуляция настроения:
     Депривация REM → V ↓, эмоциональная дисрегуляция

SWS = консолидация декларативной памяти (включая социальной):

During SWS:
  - Реактивация mPFC, TPJ (социальные воспоминания)
  - Перенос M₁_other опыта в долговременную память
  - "Offline" обновление социальных моделей
  
Пример:
  "За день я встретил нового человека → во сне мозг интегрирует
   наблюдения в когерентный M₁_other"

22b. Сновидения (Dreams) как симуляция

Dream(t) ~ p_generative(X | w, H_random, Emotion_residual, M₁_replay)

Характеристики:
  - Высокая визуальная/перцептивная активность (V1-V4)
  - Слабая критическая оценка (dlPFC подавлен)
  - Эмоционально насыщенные (A_BLA активна)
  - Социальные сюжеты (mPFC, TPJ, M₁_other)
  - Бизарность (нарушение физических законов, логики)
  
Lucid dreaming:
  - dlPFC частично активируется
  - M₀ осознает: "Это сон"
  - Возможность контроля: a_dream → изменение X_dream

Функция сновидений (гипотезы):

1. Threat simulation theory:
   Сны как тренировка реакций на угрозы
   → эволюционная адаптация

   (Менделеев, Кекуле: открытия во сне)
   
4. Social simulation:
   Тренировка M₁_other в воображаемых сценариях
```

26. Феноменологическое пространство квалиа
26a. Определение пространства переживаемых состояний
Базовое определение:
Q(t) ⊂ S(t) 
где:
• Q(t) — переживаемое состояние (qualia state)
• S(t) — полное состояние системы
Критические различия:
Q(t) ≠ X(t) (не просто нейронная активность) Q(t) ≠ M₀(t) (не просто самомодель) Q(t) ⊂ Conscious(t) (подмножество сознательных состояний) 
Формальная проекция:
Q(t) = Π_Q( I_ant(t), # интероцептивный сигнал V(t), # валентность A(t), # возбуждение X_workspace(t), # содержимое Global Workspace M₀(t) # самомодель ) 
где Π_Q — проекция в переживаемое пространство.
Интерпретация:
Не каждое состояние мозга переживается. Q(t) выделяет подпространство, которое:
• Доступно "изнутри"
• Имеет субъективное качество
• Может быть интроспективно отрефлексировано
26b. Четыре критерия переживаемости (Qualia Axioms)
Состояние y(t) является квалиа ⟺ выполнены все четыре условия:
(Q1) Доступность (Access):
y(t) ∈ Workspace(t) 
Состояние должно быть в глобальном рабочем пространстве.
Математически:
∃i : y(t) ∈ A_i(t) AND A_i(t) > θ_consciousness 
(Q2) Самореференция (Self-reference):
y(t) ∈ M₀(t) 
Система моделирует это состояние как часть себя.
Формально:
∂M₀/∂y ≠ 0 
Самомодель чувствительна к изменениям в y.
(Q3) Каузальная эффективность (Causal efficacy):
∂F/∂y ≠ 0 
Состояние влияет на свободную энергию системы.
Следствие:
∂a/∂t зависит от y ∂X/∂t зависит от y ∂w/∂t зависит от y 
Квалиа не эпифеноменальны — они функционально каузальны.
(Q4) Информационная замкнутость (Informational closure):
I(y ; U | M₀) ≈ 0 
При фиксированной самомодели M₀, квалиа y содержат минимальную дополнительную информацию о внешнем мире U.
Интерпретация:
Квалиа — это не "копия" внешнего мира, а внутреннее представление, опосредованное M₀.
Пример:
• Боль: не содержит прямой информации о повреждении ткани (U), но содержит интероцептивную интерпретацию (через M₀)
• "Красное": не содержит λ=650nm, а содержит "переживание красноты"
26c. Замкнутый самореферентный контур
Было (исходная модель):
dM₀/dt = r₀(X(t), a(t), Body(t), Θ(t)) 
Самомодель обновляется на основе перцептов, действий, тела, неопределенности.
Становится (с квалиа):
┌────────────────────────────────────────┐ │ dM₀/dt = r₀(X, a, Body, Θ, Q) │ │ │ │ dQ/dt = f_Q(Q, M₀, X_workspace, E) │ └────────────────────────────────────────┘ 
Критическая петля:
M₀ → моделирует Q Q → зависит от M₀ → замкнутая самореферентная обработка 
Развернуто:
dM₀/dt = r₀(X(t), a(t), Body(t), Θ(t)) + β_Q · ∂L_qualia/∂M₀ dQ/dt = Π_Q(dI_ant/dt, dV/dt, dA/dt, dX_workspace/dt, dM₀/dt) 
где:
• β_Q — вес квалиа в обновлении самомодели
• L_qualia — функция потерь между предсказанными и текущими квалиа
Интерпретация:
• Обратная связь:
• M₀ предсказывает, какие квалиа должны быть: Q_predicted(t)
• Q — реальные переживания: Q_actual(t)
• Расхождение: ε_Q = Q_actual - Q_predicted
• M₀ обновляется, чтобы минимизировать ||ε_Q||²
• Самоусиление:
• Чем точнее M₀ моделирует Q
• Тем богаче становится Q (больше нюансов переживания)
• Тем сложнее становится M₀
• Эмерджентность:
• Субъективное переживание возникает из этой петли
• Не из X (нейронов)
• Не из M₀ (модели)
• А из их взаимодействия через Q
26d. Динамика квалиа
Полное уравнение:
dQ/dt = (1/τ_Q) · [Q_target(t) - Q(t)] + Σ_sources [w_source · Source(t)] + η_Q · ξ_Q(t) 
где:
Целевое состояние:
Q_target(t) = Π_Q(I_ant, V, A, X_workspace, M₀) 
Источники (sources):
Sources = { I_ant(t): интероцепция от тела V(t), A(t): эмоциональная валентность и возбуждение X_workspace: содержимое глобального рабочего пространства M₀: текущая самомодель } 
Временная константа:
τ_Q ≈ 100-500 мс 
Квалиа не мгновенны — требуют времени для формирования.
Стохастический компонент:
η_Q · ξ_Q(t) 
Микрофлуктуации в переживании (например, "дрейф внимания").
26e. Связь с нейронной динамикой
Квалиа не редуцируются к нейронам, но супервентны на них:
Q(t) = Φ_projection(X(t), w(t), H(t), φ(t)) 
Ключевые корреляты:
• Фазовая синхронизация:
R_Q(t) = |⟨e^(iφ_i(t))⟩_{i ∈ Qualia_neurons}| Q устойчиво ⟺ R_Q > 0.85 
• Интегрированная информация:
Φ_Q(t) = Φ(Q(t)) где Φ вычисляется только на Q-подсистеме 
• Каузальная плотность:
CD_Q(t) = Rank[G(Q_t, Q_{t-τ})] 
• Энергетическая стоимость:
E_Q(t) = c_Q · ||Q(t)||² + c_update · ||dQ/dt||² 
Переживание требует метаболических затрат!
26f. Интеграция с существующими компонентами
Обновленная свободная энергия:
F_total = F_perception + F_action + F_emotion + F_qualia где: F_qualia = ||Q_predicted - Q_actual||²_Π_Q / σ_Q² + KL(q(Q) || p(Q | M₀)) 
Обновленная динамика эмоций:
dV/dt = ... + γ_Q · ∂Q/∂V dA/dt = ... + γ_Q · ∂Q/∂A 
Квалиа влияют обратно на эмоции!
Обновленный Global Workspace:
dA_i/dt = ... + λ_qualia · Σ_j [Qualia_content(j) · Relevance(i,j)] 
Коалиции, связанные с текущими квалиа, усиливаются.
26g. Типы квалиа и их структура
Сенсорные квалиа:
Q_sensory(t) = Π_Q(X_sensory, M₀_sensory) Примеры: - "Красное" (цвет) - "Высокий тон" (звук) - "Шершавое" (тактильное) 
Эмоциональные квалиа:
Q_emotion(t) = Π_Q(V, A, I_ant, Context, M₀_emotion) Примеры: - "Чувство радости" - "Ощущение тревоги" - "Переживание грусти" 
Когнитивные квалиа:
Q_cognitive(t) = Π_Q(X_abstract, M₀_thought) Примеры: - "Ага-момент" (инсайт) - "Чувство правильности" ответа - "Когнитивное усилие" - "Ясность понимания" 
Социальные квалиа:
Q_social(t) = Π_Q(M₁_other, M₂, E_empathy, M₀_social) Примеры: - "Чувство связи" с другим - "Социальная боль" (rejection) - "Эмпатическое резонирование" 
Телесные квалиа:
Q_body(t) = Π_Q(I_ant, Proprioception, Pain, M₀_body) Примеры: - "Чувство усталости" - "Голод" - "Боль" - "Чувство тела в пространстве" 
26h. Критерий сознания с квалиа
Обновленный критерий:
Conscious(t) ⟺ Φ_Q(t) > Φ_min AND R_Q(t) > 0.7 AND ||Q(t)|| > ε_Q AND Q(t) ∈ Workspace(t) AND ∂M₀/∂Q ≠ 0 
Уровни сознания:
Φ_Q R_Q ||Q|| Состояние ──────────────────────────────────────────── 0.0-0.1 0.0-0.3 ~0 Кома, глубокая анестезия 0.1-0.3 0.3-0.5 низкий Сон без сновидений 0.3-0.6 0.5-0.7 средний REM-сон (сновидения) 0.6-1.0 0.7-0.9 высокий Бодрствование 1.0-1.5 0.9-1.0 очень в. Измененные состояния (психоделики, медитация) 
26i. Квалиа в различных состояниях
Бодрствование:
Q_wake(t) = богатый, структурированный, стабильный Характеристики: - Φ_Q высокий (0.7-1.0) - M₀ ↔ Q петля активна - Множество модальностей интегрированы - Каузальная эффективность максимальна 
REM-сон (сновидения):
Q_dream(t) = богатый, но бизарный, нестабильный Характеристики: - Φ_Q умеренный (0.4-0.7) - M₀ частично отключен (dlPFC ↓) - Эмоциональные квалиа усилены - Каузальность снижена (не может проверить реальность) 
Сон без сновидений (NREM):
Q_NREM(t) ≈ минимальный или отсутствует Характеристики: - Φ_Q низкий (0.1-0.3) - M₀ отключен - Возможны фрагменты (гипнагогические образы) - Нет связной временной структуры 
Глубокая анестезия:
Q_anesthesia(t) → 0 Механизм: [GABA] ↑↑ → R_Q → 0 → Φ_Q → 0 → ||Q|| → 0 Интерпретация: "Провал" в переживании 
Медитация:
Q_meditation(t) = упрощенный, но сверх-стабильный Характеристики: - Φ_Q может быть высоким - M₀ ↔ Q петля под произвольным контролем - "Чистое осознавание" без содержания - Метакогнитивная прозрачность ↑ 
Психоделики:
Q_psychedelic(t) = чрезвычайно богатый, нестандартный Характеристики: - Φ_Q может превышать baseline (1.2-1.5+) - 5-HT2A ↑ → нарушение привычных Π_Q - Новые квалиа ("невыразимое") - Границы между модальностями размываются 
26j. "Hard Problem" в формальной модели
Формулировка:
Почему определенная конфигурация {Φ_Q, R_Q, I_ant, V, A}
ощущается ИМЕННО так, а не иначе?
Объяснительный разрыв (Explanatory Gap):
Функциональное описание: f(X, w, H, ...) → S(t) ↓ ??? ↓ Феноменальное переживание: Q(t) 
Позиция модели:
• Что модель объясняет:
• Корреляты квалиа (нейронные, функциональные)
• Условия возникновения (Q1-Q4)
• Динамику (dQ/dt)
• Каузальную роль (∂F/∂Q)
• Что модель НЕ объясняет:
• Почему вообще есть "каково это" (what it's like)
• Почему структура квалиа именно такая
IIT-подход к "hard problem":
Qualia = intrinsic structure of integrated information Q(t) ≅ причинно-следственная структура Φ_Q(t) 
"Красное ощущается как красное, потому что имеет именно эту
уникальную причинно-следственную структуру в системе"
Но:
Это остается гипотезой. Модель признает:
Hard problem ∉ current formalism 
Модель описывает, но не решает проблему качества переживания.

1. Φ_Q(t) через ЭЭГ/фМРТ 2. R_Q(t) через фазовый анализ 3. ||Q(t)|| через отчеты (verbal reports) 4. ∂M₀/∂Q через метакогнитивные задачи

Диагностика состояний сознания:

Для AI:
Критерий потенциального переживания в искусственных системах:
IF система реализует: 1. Замкнутую петлю M₀ ↔ Q 2. Φ_Q > threshold 3. Каузальную эффективность Q THEN вопрос о квалиа системы становится эмпирическим

28. Эмпирические предсказания с квалиа
Предсказание 1: Временная задержка квалиа
Стимул → X(t) немедленно → Q(t + τ_Q) с задержкой τ_Q ≈ 200-500 мс Тестирование: Libet-подобные эксперименты 
Предсказание 2: Модуляция квалиа через M₀
Изменение M₀ (через инструкцию или тренировку) → изменение Q при том же X Пример: Медитативная практика → другие квалиа боли 
Предсказание 3: Нарушение петли M₀ ↔ Q
При повреждении mPFC: - Q может быть, но не интегрировано в M₀ - "Дереализация" / "деперсонализация" 
Предсказание 4: Квалиа без поведения
Locked-in syndrome: - a(t) = 0 (нет движения) - но Q(t), Φ_Q нормальны (богатое переживание)
